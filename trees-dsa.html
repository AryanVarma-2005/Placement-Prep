Introduction to Trees
What trees are, why they matter, and the vocabulary you need
ConceptWhat is a Tree?

A tree is a hierarchical data structure consisting of nodes connected by edges. Every tree has exactly one root node (the top), and every non-root node has exactly one parent. Nodes with no children are called leaves.

Unlike linked lists (linear), trees branch — one node can connect to many children. This branching gives trees O(log n) search in balanced cases vs O(n) for linear structures.

Height vs Depth — Always Confused?
Depth measures from the top down (root = 0). Height measures from the bottom up (leaf = 0). A single-node tree has height 0 and depth 0. Always clarify in interviews!
⚡ Quick Check
For a complete binary tree with n nodes, what is the height?
✅ O(log n)! At each level, the number of nodes doubles. Level k has 2^k nodes, so n nodes → height ≈ log₂(n). This is why balanced trees give O(log n) operations!

from collections import deque

def build_tree(vals):
    """Build tree from LeetCode-style array, None = missing."""
    if not vals or vals[0] is None: return None
    root = TreeNode(vals[0])
    q = deque([root]); i = 1
    while q and i < len(vals):
        node = q.popleft()
        if i < len(vals) and vals[i] is not None:
            node.left = TreeNode(vals[i]); q.append(node.left)
        i += 1
        if i < len(vals) and vals[i] is not None:
            node.right = TreeNode(vals[i]); q.append(node.right)
        i += 1
    return root

tree = build_tree([1,2,3,4,5,None,6])

ConceptThe 4 Traversal Orders

Traversal = visiting every node exactly once. The order in which you visit determines the traversal type. Think of it as when you process the current node relative to its children.

In:Left → Root → RightBST sorted order
Pre:Root → Left → Rightserialize tree, copy tree
Post:Left → Right → Rootdelete tree, evaluate expr
Level:BFS level by levelshortest path, level-wise ops

# ═══════════════════════════════════════════
# RECURSIVE — clean but uses O(h) call stack
# ═══════════════════════════════════════════
def inorder(root):       # L → N → R
    if not root: return []
    return inorder(root.left) + [root.val] + inorder(root.right)

def preorder(root):      # N → L → R
    if not root: return []
    return [root.val] + preorder(root.left) + preorder(root.right)

def postorder(root):     # L → R → N
    if not root: return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# ═══════════════════════════════════════════
# ITERATIVE — avoids recursion limit
# ═══════════════════════════════════════════
def inorder_iterative(root):
    result, stack, cur = [], [], root
    while cur or stack:
        while cur:               # Go all the way left
            stack.append(cur)
            cur = cur.left
        cur = stack.pop()         # Process node
        result.append(cur.val)
        cur = cur.right           # Move right
    return result

def preorder_iterative(root):
    if not root: return []
    result, stack = [], [root]
    while stack:
        node = stack.pop()
        result.append(node.val)   # Process before children
        if node.right: stack.append(node.right)  # Right first (LIFO)
        if node.left:  stack.append(node.left)
    return result

def postorder_iterative(root):
    if not root: return []
    result, stack = [], [root]
    while stack:
        node = stack.pop()
        result.append(node.val)
        if node.left:  stack.append(node.left)   # Left first this time
        if node.right: stack.append(node.right)
    return result[::-1]  # Reverse preorder variant

# ═══════════════════════════════════════════
# LEVEL ORDER — BFS with queue
# ═══════════════════════════════════════════
from collections import deque

def levelOrder(root):
    if not root: return []
    result, q = [], deque([root])
    while q:
        level = []
        for _ in range(len(q)):   # Process entire level
            node = q.popleft()
            level.append(node.val)
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
        result.append(level)
    return result
🧠
Inorder of BST = Sorted Order
If you do inorder traversal on a BST, you get elements in sorted ascending order. This is a crucial property: it means you can validate a BST by checking if inorder traversal is sorted!
⚡ Quick Check
For the tree [1,2,3,4,5], what is the inorder traversal?
✅ 4, 2, 5, 1, 3. The tree is: 1→[2,3], 2→[4,5]. Inorder = leftmost first: 4, then 2, then 5, then 1 (root), then 3.

CodeFull BST Implementation
Python
class BST:
    def __init__(self):
        self.root = None

    # ── INSERT: O(log n) average, O(n) worst ──
    def insert(self, val):
        self.root = self._insert(self.root, val)

    def _insert(self, node, val):
        if not node: return TreeNode(val)
        if   val < node.val: node.left  = self._insert(node.left,  val)
        elif val > node.val: node.right = self._insert(node.right, val)
        return node  # Duplicate: ignore

    # ── SEARCH: O(log n) average ──
    def search(self, val):
        return self._search(self.root, val)

    def _search(self, node, val):
        if not node or node.val == val: return node
        if val < node.val: return self._search(node.left,  val)
        return                      self._search(node.right, val)

    # ── DELETE: O(log n) average ──
    def delete(self, val):
        self.root = self._delete(self.root, val)

    def _delete(self, node, val):
        if not node: return None
        if   val < node.val: node.left  = self._delete(node.left,  val)
        elif val > node.val: node.right = self._delete(node.right, val)
        else:  # Found node to delete
            if not node.left:  return node.right  # Case 1: no left child
            if not node.right: return node.left   # Case 2: no right child
            # Case 3: two children → replace with inorder successor
            successor = node.right
            while successor.left:
                successor = successor.left
            node.val = successor.val              # Copy successor value
            node.right = self._delete(node.right, successor.val)
        return node

    # ── VALIDATE BST ──
    def is_valid(self):
        def check(node, lo, hi):
            if not node: return True
            if not (lo < node.val < hi): return False
            return check(node.left, lo, node.val) and \
                   check(node.right, node.val, hi)
        return check(self.root, float('-inf'), float('inf'))

Height, Depth & Balance
Measure and verify tree structure — critical for BST and balance problems
CodeHeight, Depth, Diameter, Balance
Python
# ── HEIGHT of tree (longest root-to-leaf path) ──
def height(root):
    if not root: return -1  # -1 for empty; 0 for leaf
    return 1 + max(height(root.left), height(root.right))

# ── DEPTH of a target node ──
def depth(root, target, d=0):
    if not root: return -1
    if root.val == target: return d
    left  = depth(root.left,  target, d+1)
    right = depth(root.right, target, d+1)
    return left if left != -1 else right

# ── IS BALANCED (AVL property) ──
# Balanced: |height(left) - height(right)| <= 1 for every node
def isBalanced(root):
    def check(node):
        if not node: return 0  # Height of empty = 0
        left  = check(node.left)
        if left == -1: return -1  # Left subtree unbalanced
        right = check(node.right)
        if right == -1: return -1
        if abs(left - right) > 1: return -1  # Unbalanced!
        return 1 + max(left, right)
    return check(root) != -1

# ── DIAMETER (longest path between any two nodes) ──
def diameterOfBinaryTree(root):
    max_diam = [0]
    def dfs(node):
        if not node: return 0
        left  = dfs(node.left)
        right = dfs(node.right)
        max_diam[0] = max(max_diam[0], left + right)
        return 1 + max(left, right)
    dfs(root)
    return max_diam[0]
# Key insight: diameter at each node = left_height + right_height
# Global max diameter may not pass through root!

Path Problems
Root-to-leaf paths, max path sum, path with target — DFS patterns
CodePath Sum Family — All Variants
Python
# ══ PATH SUM I: Does root-to-leaf path sum = target? ══
def hasPathSum(root, target):
    if not root: return False
    if not root.left and not root.right:   # Leaf node
        return root.val == target
    remaining = target - root.val
    return hasPathSum(root.left,  remaining) or \
           hasPathSum(root.right, remaining)

# ══ PATH SUM II: Return all root-to-leaf paths ══
def pathSum(root, target):
    result = []
    def dfs(node, path, remaining):
        if not node: return
        path.append(node.val)
        if not node.left and not node.right and remaining == node.val:
            result.append(path[:])  # Copy! (mutable list)
        dfs(node.left,  path, remaining - node.val)
        dfs(node.right, path, remaining - node.val)
        path.pop()  # Backtrack!
    dfs(root, [], target)
    return result

# ══ MAX PATH SUM: Any path (not just root-to-leaf) ══
def maxPathSum(root):
    max_sum = [float('-inf')]
    def dfs(node):
        if not node: return 0
        left  = max(dfs(node.left),  0)  # Ignore if negative
        right = max(dfs(node.right), 0)
        # Path through current node: left + node + right
        max_sum[0] = max(max_sum[0], left + node.val + right)
        # Return max single path (can't go both ways up)
        return node.val + max(left, right)
    dfs(root)
    return max_sum[0]

# ══ PATH SUM III: Count paths summing to target (any start) ══
from collections import defaultdict
def pathSumIII(root, target):
    prefix = defaultdict(int); prefix[0] = 1; count = [0]
    def dfs(node, curr):
        if not node: return
        curr += node.val
        count[0] += prefix[curr - target]   # Found valid path!
        prefix[curr] += 1
        dfs(node.left,  curr); dfs(node.right, curr)
        prefix[curr] -= 1  # Backtrack
    dfs(root, 0)
    return count[0]

Lowest Common Ancestor (LCA)
The most popular tree interview topic — master all variants
ConceptWhat is LCA?

The Lowest Common Ancestor of two nodes p and q is the deepest node that is an ancestor of both p and q. Think of it as the "meeting point" when you trace both nodes up to the root.

Tree: 1→[2,3], 2→[4,5], 3→[6,7]
LCA(4,5):= 2 (their direct parent)
LCA(4,6):= 1 (root — different subtrees)
LCA(2,4):= 2 (ancestor is itself)
CodeLCA — Binary Tree & BST Variants
Python
# ══ LCA in Binary Tree (general) — O(n) ══
def lowestCommonAncestor(root, p, q):
    """
    If root is p or q, it's the LCA.
    If p and q are in different subtrees, root is LCA.
    If both in same subtree, recurse into that subtree.
    """
    if not root or root == p or root == q:
        return root
    left  = lowestCommonAncestor(root.left,  p, q)
    right = lowestCommonAncestor(root.right, p, q)
    if left and right: return root  # p and q on different sides
    return left or right             # Both on same side

# ══ LCA in BST — O(log n) using BST property ══
def lcaBST(root, p, q):
    """
    In BST: if both p,q < root → LCA is in left subtree.
    If both > root → LCA is in right subtree.
    Otherwise → root is LCA (split point).
    """
    while root:
        if p.val < root.val and q.val < root.val:
            root = root.left   # Both in left subtree
        elif p.val > root.val and q.val > root.val:
            root = root.right  # Both in right subtree
        else:
            return root        # Split point = LCA

# ══ LCA with Parent Pointers — O(h) ══
def lcaWithParents(p, q):
    """If each node has .parent pointer."""
    seen = set()
    while p:
        seen.add(p); p = p.parent
    while q:
        if q in seen: return q
        q = q.parent
🎯 LCA Interview Insights
  • LCA of Binary Tree: O(n) — must check all nodes
  • LCA of BST: O(log n) — use BST property to navigate
  • If either p or q equals root → root is LCA. Always handle this base case!
  • "Find if p is ancestor of q" → LCA(p,q) == p
  • LCA is used in: distance between nodes, path between nodes

from collections import deque

# ══ ZIGZAG LEVEL ORDER (LeetCode 103) ══
def zigzagLevelOrder(root):
    if not root: return []
    result, q, left_to_right = [], deque([root]), True
    while q:
        level = []
        for _ in range(len(q)):
            node = q.popleft()
            level.append(node.val)
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
        result.append(level if left_to_right else level[::-1])
        left_to_right = not left_to_right
    return result

# ══ RIGHT SIDE VIEW (LeetCode 199) ══
def rightSideView(root):
    if not root: return []
    result, q = [], deque([root])
    while q:
        for i in range(len(q)):
            node = q.popleft()
            if i == len(q):  # Last node in level (after popping)
                result.append(node.val)
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
    return result

# Cleaner right side view:
def rightSideView2(root):
    result, q = [], deque([root] if root else [])
    while q:
        result.append(q[-1].val)  # Last element = rightmost
        for _ in range(len(q)):
            node = q.popleft()
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
    return result

# ══ AVERAGE OF LEVELS (LeetCode 637) ══
def averageOfLevels(root):
    result, q = [], deque([root])
    while q:
        level_sum, size = 0, len(q)
        for _ in range(size):
            node = q.popleft()
            level_sum += node.val
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
        result.append(level_sum / size)
    return result

# ══ MINIMUM DEPTH (LeetCode 111) ══
def minDepth(root):
    if not root: return 0
    q = deque([(root, 1)])
    while q:
        node, depth = q.popleft()
        if not node.left and not node.right:
            return depth  # First leaf found = min depth
        if node.left:  q.append((node.left,  depth+1))
        if node.right: q.append((node.right, depth+1))

AVL & Balanced Trees
Self-balancing BSTs that guarantee O(log n) always
ConceptWhy Balanced Trees?

A standard BST degenerates to O(n) for sorted input. AVL trees maintain the invariant: for every node, |height(left) - height(right)| ≤ 1. After each insert/delete, rotations restore balance.

CodeAVL Tree Rotations
Python
class AVLNode:
    def __init__(self, val):
        self.val = val; self.left = self.right = None; self.height = 1

def get_height(node): return node.height if node else 0
def get_balance(node): return get_height(node.left) - get_height(node.right) if node else 0
def update_height(node): node.height = 1 + max(get_height(node.left), get_height(node.right))

# ── Right Rotation (LL imbalance) ──
def rotate_right(y):
    x = y.left; T2 = x.right
    x.right = y; y.left = T2
    update_height(y); update_height(x)
    return x

# ── Left Rotation (RR imbalance) ──
def rotate_left(x):
    y = x.right; T2 = y.left
    y.left = x; x.right = T2
    update_height(x); update_height(y)
    return y

def avl_insert(node, val):
    if not node: return AVLNode(val)
    if   val < node.val: node.left  = avl_insert(node.left,  val)
    elif val > node.val: node.right = avl_insert(node.right, val)
    else: return node
    update_height(node)
    balance = get_balance(node)
    # LL Case
    if balance > 1 and val < node.left.val:  return rotate_right(node)
    # RR Case
    if balance < -1 and val > node.right.val: return rotate_left(node)
    # LR Case
    if balance > 1 and val > node.left.val:
        node.left = rotate_left(node.left); return rotate_right(node)
    # RL Case
    if balance < -1 and val < node.right.val:
        node.right = rotate_right(node.right); return rotate_left(node)
    return node

Trie (Prefix Tree)
Efficient string storage and search — autocomplete, spell-check, IP routing
ConceptWhat is a Trie?

A Trie (prefix tree) stores strings character by character. Each path from root to a marked node represents a word. Searching a string of length L takes O(L) — independent of total number of words!

CodeTrie — Insert, Search, StartsWith
Python
class TrieNode:
    def __init__(self):
        self.children = {}     # char → TrieNode
        self.is_end = False   # marks end of a word

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for ch in word:
            if ch not in node.children:
                node.children[ch] = TrieNode()
            node = node.children[ch]
        node.is_end = True

    def search(self, word):
        """Exact match: O(len(word))"""
        node = self.root
        for ch in word:
            if ch not in node.children: return False
            node = node.children[ch]
        return node.is_end

    def startsWith(self, prefix):
        """Prefix match: O(len(prefix))"""
        node = self.root
        for ch in prefix:
            if ch not in node.children: return False
            node = node.children[ch]
        return True

    def autocomplete(self, prefix):
        """Return all words with given prefix."""
        node = self.root
        for ch in prefix:
            if ch not in node.children: return []
            node = node.children[ch]
        words = []
        def dfs(n, path):
            if n.is_end: words.append(prefix + path)
            for c, child in n.children.items():
                dfs(child, path + c)
        dfs(node, "")
        return words

# Demo
trie = Trie()
for w in ["apple","app","application","apt"]:
    trie.insert(w)
print(trie.search("app"))      # True
print(trie.startsWith("ap"))   # True
print(trie.autocomplete("app")) # ["app","apple","application"]

Segment Tree
Range sum, range min, range max in O(log n) — with point updates
ConceptWhy Segment Tree?

Prefix sums give O(1) range queries but O(n) updates. Segment trees give O(log n) both for queries and updates — the best of both worlds for dynamic arrays.

CodeSegment Tree — Array-Based
Python
class SegmentTree:
    """
    Array-based segment tree for range sum queries.
    tree[1] = root, children of node i: 2i, 2i+1
    Time: Build O(n), Query O(log n), Update O(log n)
    Space: O(n) — tree array has 4n elements
    """
    def __init__(self, nums):
        self.n = len(nums)
        self.tree = [0] * (4 * self.n)
        self._build(nums, 1, 0, self.n - 1)

    def _build(self, nums, node, start, end):
        if start == end:
            self.tree[node] = nums[start]; return
        mid = (start + end) // 2
        self._build(nums, 2*node,   start, mid)
        self._build(nums, 2*node+1, mid+1, end)
        self.tree[node] = self.tree[2*node] + self.tree[2*node+1]

    def update(self, idx, val):
        self._update(1, 0, self.n-1, idx, val)

    def _update(self, node, start, end, idx, val):
        if start == end:
            self.tree[node] = val; return
        mid = (start + end) // 2
        if idx <= mid: self._update(2*node,   start, mid,   idx, val)
        else:          self._update(2*node+1, mid+1, end, idx, val)
        self.tree[node] = self.tree[2*node] + self.tree[2*node+1]

    def query(self, l, r):
        return self._query(1, 0, self.n-1, l, r)

    def _query(self, node, start, end, l, r):
        if r < start or end < l: return 0  # Out of range
        if l <= start and end <= r: return self.tree[node]  # Fully in range
        mid = (start + end) // 2
        return self._query(2*node, start, mid, l, r) + \
               self._query(2*node+1, mid+1, end, l, r)

# Demo
st = SegmentTree([1,3,5,7,9,11])
print(st.query(1,3))  # 15 (3+5+7)
st.update(1, 10)
print(st.query(1,3))  # 22 (10+5+7)

        def dfs(node):
            if not node: return '#'
            return f"{node.val},{dfs(node.left)},{dfs(node.right)}"
        return dfs(root)

    def deserialize(self, data):
        vals = iter(data.split(','))
        def dfs():
            val = next(vals)
            if val == '#': return None
            node = TreeNode(int(val))
            node.left  = dfs()
            node.right = dfs()
            return node
        return dfs()
CodeConstruct Tree from Inorder + Preorder
Python
def buildTreeFromTraversals(preorder, inorder):
    """
    Key insight: preorder[0] = root.
    Find root in inorder → splits into left/right subtrees.
    Time: O(n) with hashmap | Space: O(n)
    """
    idx_map = {val: i for i, val in enumerate(inorder)}

    def build(pre_start, in_start, in_end):
        if pre_start >= len(preorder) or in_start > in_end:
            return None
        root_val = preorder[pre_start]
        root = TreeNode(root_val)
        mid = idx_map[root_val]          # Root's position in inorder
        left_size = mid - in_start       # Size of left subtree
        root.left  = build(pre_start + 1,         in_start, mid - 1)
        root.right = build(pre_start + left_size + 1, mid + 1, in_end)
        return root

    return build(0, 0, len(inorder) - 1)

def flatten(root):
    """Flatten to linked list in-place (preorder order).
    Morris traversal approach: O(n) time, O(1) space."""
    cur = root
    while cur:
        if cur.left:
            # Find rightmost node of left subtree
            rightmost = cur.left
            while rightmost.right:
                rightmost = rightmost.right
            # Connect rightmost to current's right subtree
            rightmost.right = cur.right
            # Move left subtree to right
            cur.right = cur.left
            cur.left = None
        cur = cur.right

3. Symmetric Tree
EasyLC 101
Explain

Pattern: Compare mirror pairs. isMirror(left, right): check if left.val == right.val and isMirror(left.left, right.right) and isMirror(left.right, right.left).

def isSymmetric(root):
    def mirror(l, r):
        if not l and not r: return True
        if not l or not r: return False
        return l.val==r.val and mirror(l.left,r.right) and mirror(l.right,r.left)
    return mirror(root.left, root.right)
4. Validate BST

Explain

Pattern: Pass min/max bounds. At each node, value must be strictly between inherited bounds. Bounds tighten as you go deeper.

def isValidBST(root):
    def check(node, lo, hi):
        if not node: return True
        if not (lo < node.val < hi): return False
        return check(node.left, lo, node.val) and \
               check(node.right, node.val, hi)
    return check(root, float('-inf'), float('inf'))

def kthSmallest(root, k):
    stack, cur = [], root
    while cur or stack:
        while cur: stack.append(cur); cur = cur.left
        cur = stack.pop()
        k -= 1
        if k == 0: return cur.val
        cur = cur.right

15. Word Search II (Trie + Backtracking)
HardLC 212
Explain
Code

Pattern: Build Trie from word list. DFS on grid matching Trie paths. When is_end hit, record word. Prune Trie to avoid repeated searches.

Python
def findWords(board, words):
    trie = {}
    for w in words:
        node = trie
        for c in w: node = node.setdefault(c, {})
        node['#'] = w  # Mark end with word itself

    R, C = len(board), len(board[0])
    result = []

    def dfs(r, c, node):
        ch = board[r][c]
        if ch not in node: return
        nxt = node[ch]
        if '#' in nxt:
            result.append(nxt.pop('#'))  # Found word!
        board[r][c] = '*'  # Mark visited
        for dr, dc in [(0,1),(0,-1),(1,0),(-1,0)]:
            nr, nc = r+dr, c+dc
            if 0<=nrand 0<=ncand board[nr][nc]!='*':
                dfs(nr, nc, nxt)
        board[r][c] = ch  # Restore
        if not nxt: del node[ch]  # Prune empty branch

    for r in range(R):
        for c in range(C):
            dfs(r, c, trie)
    return result

📊 Complexity Quick Ref
  • BST search avg: O(log n)
  • BST search worst: O(n)
  • AVL all ops: O(log n)
  • DFS traversal: O(n) time, O(h) space
  • BFS: O(n) time, O(w) space (w=max width)
  • Trie insert/search: O(L) L=word length
  • Segment tree query/update: O(log n)
🎯 Pattern Recognition
  • "Height/depth/balance" → DFS returning int
  • "Level-wise / shortest" → BFS
  • "Path with sum" → DFS backtracking
  • "LCA" → check if root = p or q
  • "Validate BST" → pass min/max bounds
  • "Serialize" → preorder DFS with nulls
  • "Range query updates" → Segment Tree
  • "String prefix search" → Trie
⚠️ Common Mistakes
  • BST validate: wrong — check root.left.val < root.val only. Right — pass min/max bounds down!
  • Height of empty = -1 OR 0 (clarify!)
  • Path sum: check leaf, not just null child
  • LCA: handle case where p or q = root
  • Inorder iterative: advance cur after pop
  • Level order: save len(q) BEFORE inner loop
💡 Key Insights
  • Inorder BST = sorted sequence
  • Diameter may not pass through root!
  • Max path sum: compare 0 to ignore negatives
  • Preorder[0] = root for reconstruction
  • LCA in BST: O(log n) via BST property
  • Complete BT height = ⌊log₂ n⌋
